# Fatec Projeto Scripts - Neto / 5º Semestre

**Description:** Projeto do professor Neto / 5º Semestre / Scripts

**Installation:**

1.  Clone the repository: `git clone https://github.com/fatec-projeto-scripts-neto.git`
2.  Navigate to the repository: `cd fatec-projeto-scripts-neto`
3.  Install dependencies: `npm install`
4.  Set up .vscode: `npm install -g vsc-extension`
5.  Set up nodemon: `npm run nodemon`
6.  Initialize package.json: `npm run init`

**Usage:**

*   **app.js:**  Core application logic.  This file handles all application requests and interactions.
*   **api/:**  Handles API requests.  This includes routes, data fetching, and API endpoints.
*   **index.ejs:**  Main entry point for the application.
*   **aula-hoje:**  Displays the current lesson.
*   **config/:**  Configuration settings for the application.
*   **database/:**  Database connection details and data management.
*   **package.json:**  Defines project dependencies, scripts, and configurations.
