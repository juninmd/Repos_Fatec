# Fatec-Ionic-Lojinha

**Description:** Trabalho do professor fausto, 6º Semestre - Tópicos Especiais 2

This repository contains the source code for the Fatec-Ionic-Lojinha project.

## Installation

1.  Clone the repository: `git clone https://github.com/fatec-students/fatec-ionic-lojinha.git`
2.  Navigate to the project directory: `cd fatec-ionic-lojinha`
3.  Install dependencies: `npm install`
4.  Run the development server: `npm start`

## Usage

The project is structured as follows:

*   **src:** Contains the core application logic, UI components, and data structures.
*   **tsconfig.json:** Configuration for the TypeScript compiler.
*   **tslint.json:** Code linting rules.
*   **package.json:** Defines project dependencies and scripts.
*   **ionic.config.json:** Configuration for the Ionic framework.
*   **livros desenvolvimento mobile:** Contains supplementary learning materials.
*   **FLORIANO (2016). Começando com Ionic 2.pdf:**  A helpful guide for Ionic 2.
*   **HOC (2016). Ionic 2 Cookbook.pdf:**  Provides practical tips and code examples for Ionic 2.
*   **LICENSE:**  License information for the project.
*   **package-lock.json:**  Lockfile for package dependencies.
*   **yarn.lock:**  Lockfile for package dependencies.

To start development, navigate to the `src` directory and run `npm start`.  This will start the development server and allow you to interact with the application.
