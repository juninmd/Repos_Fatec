# Fatec-Compare-C

**Description:** Comparação entre dois números se são equivalentes reversamente.

**Installation:**

1.  Clone the repository: `git clone https://github.com/your-username/fatec-compare-c.git`
2.  Install dependencies: `npm install` (assuming Makefile.win exists).
3.  Configure: `npm config set rel=production` (if applicable).

**Usage:**

*   **Generate a Comparison:**  Run `make compare.exe` to generate the comparison output.
*   **Interpret Output:** The comparison output will indicate if two numbers are equivalent reversely.  The output format is designed for easy parsing.
